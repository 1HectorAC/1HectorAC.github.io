4. Color Mixer

4.a ColorController.cs

The controller contains both a GET and POST version of the Create view action. The GET version was pretty simple and doesn't require much explaing. The POST one contained more important parts to mention. The logic in this part involved splitting the hexadecimal value into its three parts and then converting the values into a double value, adding the red, blue, and green of the two inputted colors, adding them, and then converting it back. Something to note is that I had a check to convert it to a six digit hexadecimal if it was three digits. It also had some form validation in this part in terms of if the hexadecimal input value was 4 or 5 digits long (not counting the #) then it wouldn't do anything. This was necessary since it was the only exception in the range limit I set on the html validation that was valid.
